# Array of Objects

## Concept
Just like arrays of primitive data types (int, float), we can create arrays of user-defined data types (objects).

## Syntax
```cpp
ClassName arrayName[Size];
```

## Usage
- Useful when we need to store data for multiple entities (e.g., 50 students, 10 employees).
- Each element in the array is an object of the class.
- Accessed using index: `arrayName[i].memberFunction()`.

## Initialization
- If the class has a constructor, the array initialization can be done using an initializer list.
